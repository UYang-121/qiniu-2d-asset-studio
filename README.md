# 2D 素材生成工具

> Demo 视频链接：待补充

一个用于生成 2D 游戏基础素材草稿的前端工具，支持批量预览、单张素材下载，以及图集与坐标文件导出。

## 功能说明

- 支持三类素材：`道具`、`地块`、`图标`
- 支持五种场景主题：`森林`、`沙漠`、`冰雪`、`熔岩`、`赛博`
- 支持三种视觉风格：`像素风`、`扁平风`、`科技风`
- 支持通过补充描述调整当前批次主配色
- 支持单张素材导出为 SVG
- 支持整批导出图集 PNG 和 atlas JSON

## 本地运行

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
```
